# Калькулятор Расходов (Food Calc)

Простой и удобный веб-инструмент для расчета расходов в кафе и ресторанах для группы друзей.

## Возможности

*   **Учет участников:** Добавление имен участников.
*   **Личные заказы:** Добавление блюд для каждого участника.
*   **Общие блюда:** Добавление блюд, которые делятся на всех (например, пицца, закуски).
*   **Автоматический расчет:**
    *   Личные расходы каждого участника.
    *   Распределение стоимости общих блюд поровну.
    *   Учет процента за обслуживание (комиссия).
    *   Итоговая сумма чека.
*   **История чеков:** Сохранение и просмотр прошлых расчетов (используется LocalStorage браузера).
*   **Скриншот чека:** Возможность сделать красивый скриншот итогового расчета для отправки друзьям.
*   **Темная/Светлая тема:** Переключение интерфейса.
*   **Управление плательщиком:** Указание того, кто оплатил счет, и расчет долгов остальных участников перед ним.
*   **Копирование реквизитов:** Быстрое копирование номера карты/кошелька плательщика.

## Как использовать

1.  Откройте сайт или файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari, Edge).
2.  **Добавьте участников:** Нажмите "Блюдо" и введите имя участника и название блюда.
3.  **Добавьте общие блюда:** Нажмите "Общее" для добавления блюд на всех.
4.  **Укажите сервис:** Введите процент обслуживания в поле "Сервис (%)" и нажмите галочку.
5.  **Выберите плательщика:** В поле "Плательщик" выберите того, кто оплатил счет.
6.  **Укажите реквизиты:** (Опционально) Введите номер карты/кошелька для возврата долга.
7.  **Скриншот:** Нажмите кнопку "Скриншот", чтобы получить картинку с расчетом.

## Технологии

*   HTML5
*   CSS3 (Flexbox, Grid, CSS Variables)
*   JavaScript (ES6+)
*   Font Awesome (иконки)
*   html2canvas (для создания скриншотов)
*   Google Fonts (Inter, Montserrat)

## Установка

Просто скачайте файл `index.html` и откройте его. Интернета требует только для загрузки шрифтов и иконок (CDN).
